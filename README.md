Java / Maven Kickstart Project
==============================

Installation
------------

You may fork this project using the github functionality

Usage
-----

    git clone git://github.com/aschi/ch.hszt.Tierverwaltung.git
    cd ch.hszt.Tierverwaltung
    mvn install

Features
--------

* Java 1.6
* Maven
* Junit
* Mock Objects
* slf4j
